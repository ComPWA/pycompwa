class StateTransitionGraph[source]¶

Bases: object

Graph class which contains edges and nodes, similar to feynman graphs.

The graphs are directed, meaning the edges are ingoing and outgoing to specific nodes (since feynman graphs also have a time axis) This class can contain the full information of a state transition from a initial state to a final state. This information can be attached to the nodes and edges via properties.
