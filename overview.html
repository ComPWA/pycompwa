About pycompwa¶

pycompwa is a collection of python modules and the python interface to ComPWA. ComPWA stands for “Common Partial Wave Analysis framework”.

Analogous to ComPWA, pycompwa’s focus is also modularity. pycompwa extends the ComPWA framework with some useful modules:

  • expertsystem
    An amplitude/intensity model generator

  • ui
    The python interface to ComPWA

  • plotting
    Facilitates the visualization of data and results
